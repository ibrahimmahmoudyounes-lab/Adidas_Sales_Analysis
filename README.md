## Project Overview
This project involves a comprehensive analysis of **Adidas Sales** data using **Microsoft Excel**. The goal is to identify sales trends, regional performance, and product profitability.

## Features
* **Data Cleaning:** Processed raw sales data for accuracy.
* **Pivot Tables:** Summarized key metrics (Total Sales, Operating Profit).
* **Interactive Dashboard:** Visualized insights using Excel charts and slicers.

## Project Files
* `Adidas US Sales Datasets.xlsx`: Raw data.
* `Adidas_Sales_Analysis.xlsm`: The analysis and main dashboard.
* `Adidas_Sales_Dashboard.PNG`: Screenshot of the final dashboard.
